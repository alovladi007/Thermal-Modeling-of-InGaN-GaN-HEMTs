VIM NAVIGATION:
==============
- gt or :tabn - Next tab
- gT or :tabp - Previous tab
- :tabs - List all tabs
- [tab number]gt - Go to specific tab
- :q - Close current tab
- :qa - Close all and exit
- :w - Save current file
- :wa - Save all files
